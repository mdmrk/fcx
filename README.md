Userscript para mejorar la experiencia de navegación en ForoCoches, eliminando publicidad y añadiendo funcionalidades útiles.

### Funcionalidades

- [x] **Scroll Infinito**: Carga automáticamente la siguiente página de hilos sin tener que hacer clic en "Siguiente".
- [x] **Sin Publicidad**: Elimina banners y anuncios molestos.
- [x] **Interfaz Limpia**: Opción para ocultar la barra lateral (en el diseño nuevo) y ganar espacio de lectura.
- [x] **Mejoras Visuales**: Estilizado de comentarios y avatares para una lectura más cómoda.
- [x] **Configurable**: Menú de ajustes integrado para activar/desactivar funciones según tus preferencias.

<br />

### Cómo usar

1. Instala una extensión de gestor de userscripts como **Tampermonkey** o **Violentmonkey**.
2. Instala este script.
3. Navega por [forocoches.com](https://forocoches.com) y disfruta de las mejoras.
